1｜JSTで置く

観測断面では、東京の OpenAI Builder Event は以下で動いている。

OpenAI Builder Event Tokyo
予定時刻：2026/06/16 17:00 JST
キャンセル告知：イベント約1時間前
つまり告知時刻：2026/06/16 16:00 JST

これをサンフランシスコ時間に直す。

2026/06/16 16:00 JST
= 2026/06/16 00:00 PDT

つまり、日本側が「1時間前に急遽キャンセル」と受け取った時刻は、Sam側では6/16深夜0時である。

この時点で病院にいるビデオがある。

しかし、東京イベントに17:00 JSTで登壇するには、SFOから東京へ身体を運ぶ必要がある。

2｜SFOから東京はもう死んでいる

SFOから東京へ移動する場合、最低でも以下が必要になる。

病院 / 自宅 / 滞在先 → 空港移動
出国・保安・搭乗
SFO → HND / NRT 長距離飛行
日本入国
空港 → 都内会場移動
会場入り
登壇準備

飛行機だけでも約10〜11時間級。
その他の移動・手続き・会場入りを含めれば、現実には13〜16時間級の導線になる。

したがって、6/16 17:00 JST に東京会場へ立つには、少なくとも前日〜当日早朝JST、PDTでは6/15の昼〜夕方以前には移動判断が終わっていなければならない。

つまり、

6/16 16:00 JST に病院にいる
= その1時間後の東京登壇は物理的に不可能

ここで起きているのは「1時間前に突然、東京移動が不可能になった」ではない。

正確には、

東京登壇はもっと前に物理的に詰んでいた。
しかし、日本側への通知が1時間前に落ちた。

である。
